JQuery 小结

Nov 20th, 2014

Introduction

JQuery相比于原生的javascript，提供了更简洁，优美的函数接口用来操作dom，并且屏蔽了大多数主流浏览器的差异性。

JQuery的基本思想就是通过$来选择dom中的元素，并在此基础提供一系列的dom travesal函数用来选择需要操作的元素，在通过一系列的方法来操作选中的dom元素。比如改变他的attributes, css property, 给选定的元素绑定bind面对给这事件的callback方法。

Part 1 find what you want to

JQuery为我们提供了money，哦不$函数来选择我们需要的dom元素。$函数的接受string做为输入，返回一个JQuery collection。

dom其实就是数据结构中的树结构，JQuery还提供给我们一些十分简洁的方法来在树中遍历traverse.

小TIP

Part 2 Manipulate what you find

一些常用的方法。

  • toggleClass(someClass)为如果元素没有someClass，则加上，如果有，则去除对应的someClass
  • attr() 即是一个getter，也是一个setter。用来操纵所选dom元素的属性，如href，src等等
  • text(), html()获取元素内部的text或html内容

关于对减少元素的方法

对dom树中增加元素的方法
